## Pré-requisitos

Git: https://git-scm.com/ <br>
Node.js: https://nodejs.org/en/<br>

## Instalação

```
git clone <repo url>
cd WEB
npm install
npm start
```

## Storybook

Storybook é uma lib que permite visualizar isoladamente os componentes desenvolvidos.

Para executar basta rodar o comando abaixo:

```
npm storybook
```
